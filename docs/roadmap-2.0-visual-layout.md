# Visual layout editor → 2.0.0

The WYSIWYG visual layout editor for the light card (and related `nodalia-visual-layout-editor.js` work) is **not** part of the current `1.2.0-alpha` line.

## Branch

All commits through **`1.2.0-alpha.20`** (editor, live preview, grid guides, palette thumbnails, etc.) are preserved on:

**`future/2.0.0-visual-layout`**

Use that branch when resuming work toward a **`2.0.0`** release.

## Current alpha line (`main` baseline)

After **`1.2.0-alpha.21`**, `alpha` tracks **`main`** (stable **1.1.4**) and targets smaller, incremental features first — e.g. a climate card popup for setpoint schedules.

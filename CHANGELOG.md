# 📦 Changelog

All notable changes to **Nodalia Cards** will be documented in this file.

The format is inspired by Keep a Changelog and semantic versioning principles.

---

## [0.1.0] - Initial stable release

### ✨ Added
- First stable release of Nodalia Cards
- Core set of custom cards:
  - Navigation bar
  - Media player
  - Light, fan and humidifier cards
  - Climate card with circular dial
  - Vacuum cards (basic + advanced map)
  - Weather card
  - Entity and utility cards
- Initial energy flow card
- Initial graph card
- Circular gauge card
- Alarm panel card

### 🎨 Improvements
- Consistent visual design across most cards
- Clean layout with gradients and soft shadows
- Mobile-first UI approach
- Improved spacing and hierarchy

### ⚡ UX
- Smooth animations across many cards
- State transitions implemented
- Tactile feedback (haptics) support in compatible cards
- Navigation system between views

### 📚 Docs
- Initial README with installation and examples
- Basic usage documentation for all main cards
